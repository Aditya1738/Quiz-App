# Quiz-App

#About This Project Quiz App is a web-based quiz platform built using HTML, CSS, JavaScript (jQuery). The app features a clean interface and allows users to enter their name and select from multiple quiz categories:

Pipes and Cisterns
*   Probability
*   Problems on Age
*   Profit and Loss
Each category presents a randomized set of questions with four options, tracks the timer, and displays your score at the end. The app also calculates stats like attempted questions, correct/wrong answers, and total percentage.

#Key Features:

*   Category selection for focused quizzes
*   Dynamic question & option rendering
*   Real-time score and timer display
*   Result breakdown: attempts, correct, wrong, percentage
*   "Go to Home" button for retrying/starting over
*   Visually enhanced using custom CSS and the Testbook-style logo
*   
#Technologies Used:

HTML5 for structure
CSS3 for styling
JavaScript/jQuery for interactivity
